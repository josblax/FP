### Ciencia de Datos

1. **Promedio Simple**: Pide 3 valores numéricos (ej. ventas de 3 días). Súmalos y divide el total entre 3. (¡Cuidado con los paréntesis!).

2. **Error Cuadrático**: Pide el "Valor Real" y el "Valor Predicho" por el algoritmo. Réstalos y eleva el resultado a la potencia 2.

3. **Escalado Min-Max**: Pide un valor a normalizar, el valor mínimo del dataset y el máximo. Calcula: (Valor - Minimo) / (Maximo - Minimo).

4. **Lotes de Entrenamiento (Batches)**: Pide el total de registros y el tamaño del lote. Usa TRUNC para saber cuántos lotes enteros se enviarán a la Red Neuronal.

5. **Registros Huérfanos**: Con los datos anteriores, usa MOD para saber cuántos registros sobrarán para el último lote incompleto.

6. **Tasa de Error Porcentual**: Pide el error total y el valor real total. Divide Error entre Real y multiplica por 100.

7. **Promedio Ponderado**: Pide dos calificaciones y el "peso" de cada una. Calcula: ((c1 * p1) + (c2 * p2)) / (p1 + p2).

8. **Varianza Simple (Paso 1)**: Pide dos datos y calcula su media. Resta la media al primer dato y elévalo al cuadrado. Muestra el resultado.

9. **Tasa de Crecimiento (MoM)**: Pide ventas mes pasado y actual. Calcula: (Actual - Pasado) / Pasado * 100.

10. **Conversión de Probabilidad**: Un modelo devuelve una probabilidad de 0.845. Pide el dato, multiplícalo por 100 para mostrarlo como 84.5%.
