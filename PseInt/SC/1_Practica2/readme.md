# Ingeniería en Sistemas Computacionales (ISC)
   
1. Validación de Longitud de Contraseña
Enunciado: Solicita al usuario la longitud (cantidad de caracteres) de una contraseña ingresada. Si es menor a 8, muestra "Error: Contraseña insegura (Mínimo 8 caracteres)". SiNo, "Contraseña aceptada, guardando en base de datos".
Mini-Rúbrica: Entrada de variable entera (20%) | Lógica condicional < (40%) | Mensajes de seguridad (20%) | DFD e imágenes completas (20%).

2. Autenticación de Administrador
Enunciado: Lee el nivel de privilegio de un usuario (1 para Admin, 0 para Invitado). Si es igual a 1, imprime "Acceso concedido al Root/Terminal del Servidor". SiNo, "Acceso denegado: Privilegios insuficientes".
Mini-Rúbrica: Manejo de variable booleana/entera (30%) | Uso de igualdad = (30%) | Lógica de ciberseguridad (20%) | DFD e imágenes completas (20%).

3. Latencia de Peticiones (Ping al Servidor)
Enunciado: Solicita el tiempo de respuesta de un servidor en milisegundos (ms). Si supera los 100 ms, advierte "Alerta de Timeout: Alta latencia en la red". SiNo, "Paquetes recibidos: Conexión de red estable".
Mini-Rúbrica: Captura de latencia (20%) | Condicional > (40%) | Conceptos de redes (20%) | DFD e imágenes completas (20%).

4. Espacio en Disco para Instalación de SO
Enunciado: Pide los Gigabytes (GB) libres en la partición del disco duro. Si es menor a 20, indica "Abortar instalación: Espacio insuficiente para el Sistema Operativo". SiNo, "Iniciando formateo e instalación de paquetes".
Mini-Rúbrica: Variable de almacenamiento (20%) | Lógica relacional (40%) | Textos de sistemas (20%) | DFD e imágenes completas (20%).

5. Estado de Compilación de Código
Enunciado: Pregunta la cantidad de errores de sintaxis detectados por el compilador. Si es mayor a 0, muestra "Build Failed: Revisar logs y corregir código". SiNo, "Build Successful: Archivo ejecutable (.exe) generado".
Mini-Rúbrica: Lectura de errores (20%) | Operador > (40%) | Lenguaje técnico de desarrollo (20%) | DFD e imágenes completas (20%).

6. Monitoreo de Temperatura del CPU
Enunciado: Lee la temperatura actual del microprocesador en °C. Si es mayor o igual a 90, alerta "Riesgo de Thermal Throttling: Reduciendo frecuencia del reloj (Hz)". SiNo, "Temperatura operativa en rangos normales".
Mini-Rúbrica: Entrada térmica (20%) | Condición >= (40%) | Arquitectura de computadoras (20%) | DFD e imágenes completas (20%).

7. Saturación de Memoria RAM (OOM Killer)
Enunciado: Solicita el porcentaje de uso actual de la memoria RAM del servidor. Si sobrepasa el 95%, imprime "Memoria crítica: Ejecutando Out-Of-Memory Killer para cerrar procesos". SiNo, "Asignación de memoria (Malloc) permitida".
Mini-Rúbrica: Manejo de porcentaje (20%) | Estructura condicional (40%) | Conceptos de sistemas operativos (20%) | DFD e imágenes completas (20%).

8. Reglas de Firewall (Bloqueo de Puertos)
Enunciado: Pide el número de puerto de red por el que se intenta ingresar. Si el puerto es exactamente el 22 (SSH), muestra "Tráfico permitido en el Firewall". SiNo, "Conexión rechazada: Puerto cerrado por seguridad".
Mini-Rúbrica: Comparación exacta = (30%) | Uso de Si..Entonces y SiNo (30%) | Conocimiento de redes/puertos (20%) | DFD e imágenes completas (20%).

9. Batería de Respaldo del Servidor (UPS)
Enunciado: Solicita los minutos restantes de autonomía de la UPS tras un corte eléctrico. Si es menor o igual a 5, advierte "Iniciando script de apagado seguro de bases de datos". SiNo, "Manteniendo energía de respaldo a los racks".
Mini-Rúbrica: Control de tiempo (20%) | Lógica <= (40%) | Mensajes de infraestructura IT (20%) | DFD e imágenes completas (20%).

10. Límite del Pool de Conexiones a BD
Enunciado: Lee la cantidad de usuarios conectados concurrentemente a la base de datos SQL. Si supera los 150, indica "Connection Pool lleno: Petición en cola de espera". SiNo, "Abriendo nueva conexión al servidor SQL".
Mini-Rúbrica: Variable de usuarios (20%) | Lógica SiNo (40%) | Backend y Base de datos (20%) | DFD e imágenes completas (20%).
